PseudoPy v0.1.0

Initial public web release.

Added

  • Web executor UI with runtime output and error panels.
  • Upload and save support for .ppy files.
  • Docs tab with embedded documentation page.
  • Smart editor tab/shift-tab indentation support.
  • Parser support for do, stateStr/stateInt/stateFloat/stateBool, lowercase booleans, and flexible when syntax.
  • Loop controls: stop and restart.

Fixed

  • Website runtime handling for parser/runtime errors.
  • Mirror workflow for keeping src and docs in sync.
