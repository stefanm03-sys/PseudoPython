1.0 - The Basics

1.1 - Comments

Comments write out your thoughts and are ignored by the program. You can write them on one line, or span across numerous! Single line comments start with a hashtag # or // and continue until the end of the line. Like here: 


                    # This is a single line comment in Pseudopy!
                    // This is also a single line comment in Pseudopy!
Multi line comments start with /* and //. /* comments end with */, and // comments end with // on a new line. So you can do this: 
                    
                    /*
                    This is a multi line comment in Pseudopy!
                    */

                    // This is also a multi line comment in Pseudopy!
                    // It just ends when you put another set of double slashes on a new line!

1.4 - Variables and Assignment

Variables in Pseudopy are created using the var and is operators. var is used to declare a shiny new variable, and is can used to assign a value to any variable. To start making your brand new, original variable, use var followed by the variable name, then is, and finally the value you want to assign to it. This assigns data to the variable, and you can use it later in your code by referencing the variable name. For example: 


                    var x is 10 # This declares a variable x with `10` assigned to it.
                    var name is "Alice" # This declares a variable name with "Alice" assigned to it.
                    var sum is 5 + x # This declares a variable sum with the value of 5 + x, which is 15, assigned to it.
                    state(sum) # This prints the variable sum, which outputs 15, since x is 10 and is being added with 5.
Functions are also creatable! You define a function using function, followed by the name. You can do function myFunc or function myFunc(parameters) if you want to specify parameters. Functions do not use is for creation, and instead just require you to Enter, in which you can write the function body. For example: 

                    function greet(user) # Creates a function called greet with a parameter called user
                        state("Hello there!") # Contents of the function
                    endpt # Closes the function body
